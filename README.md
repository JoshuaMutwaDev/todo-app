# ✅ Todo List App

A simple and interactive Todo List app for managing daily tasks.  
Built with **HTML, CSS, and JavaScript**.

---

## 🚀 Live Demo  
👉 [View Todo App](https://my-todo-list-app-site.netlify.app)

---

## 🛠️ Tech Stack  
- HTML  
- CSS  
- JavaScript  
- LocalStorage (to save tasks)
